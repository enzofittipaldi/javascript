OPERADORES PT1

- Aritméticos
	-  + soma
		- 5 + 2 = 7

	- - subtração
		- 5 - 2 = 3

	- * multiplicação
		- 5 * 2 = 10

	- / divisão (real)
		- 5 / 2 = 2.5

	- % resto (divisão inteira)
		- 5 % 2 = 1

	- ** potencia
		- 5 ** 2 = 25

* Ordem de precedência*
	- () -> ** -> *, /, % -> +, -
		
.
- Atribuição Simples
	- var a = 5 + 3 -> 8
	  var b = a % 5 -> 3
	  var c = 5 * b ** 2 -> 45
	  var d = 10 - a / 2 -> 6
	  var e = 6 * 2 / d -> 2
	  var f = b % e + 4 / e -> 3

- Auto Atribuições
	- var n = 3 -> 3
	  n = n + 4 -> 7
	  n = n - 5 -> 2
	  n = n * 4 -> 8
	  n = n / 2 -> 4
	  n = n ** 2 -> 16
	  n = n % 5 -> 1

- Simplificando
	- n += 4 ( só serve se a mesma variável receber ela + 4)
	  n -= 5
	  n *= 4
	  n /= 2
	  n **= 2
	  n %= 5

- Incremento
	- x = 5 -> 5
	  x = x + 1 -> 6
		- x++ ( adiciona uma unidade da variável)
		- ++x 

	  x = x - 1 -> 5
		- x-- ( subtrai uma unidade da variável)
		  --x

Módulo B - Aula 08
OPERADORES PT2

- relacionais (resultado sempre vai ser valor booleano -> verdadeiro ou falso)
	- > Maior que
		- 5 > 2 = true

	- < Menor que
		- 7 < 4 = false

	- >= Maior ou igual que
		- 8 >= 8 = true

	- <= Menor ou igual que
		- 9 <= 7 = false

	- == Igual
		- 5 == 5 = true

	- != Diferente
		- 4 != 4 = false

- Identidade
	- 5 == 5 = true
	- 5 == '5' = true
	- 5 === '5' = false ( valor de igualdade restrita ou de identidade) 
		- verifica se são do mesmo valor e do mesmo tipo

	- 5 === 5 = true

- Exemplos
	- preço >= 200.50  //o preço é m,aior ou igual a 200.50?
	- idade < 18  //a idade é menor do que 18?
	- curso == 'JavaScript'  //o curso é de JavaScript?
	- n1 != n2  //o primeiro numero é diferente do segundo?

.
- lógicos
	- ! "Negação"
		- true -> false
		- false -> true

	- && "Conjunção" ou "e"
		- true && true -> true
		- true && false -> false
		- false && true -> false
		- false && false -> false

	- || "Disjunção" ou "ou"
		- true || true -> true
		- true || false -> true
		- false || true -> true
		- false || false -> false

* Ordem de precedência*
	- Operadores aritméticos -> operadores relacionais -> operadores lógicos 
	- ! -> && -> ||

- Exemplos
	- idade >= 15 && idade <= 17  //a idade está entre 15 e 17?
	- estado == 'RJ' || estado == 'SP'  //o estado é RJ ou SP?
	- salario > 1500 && sexo != 'M'  //o salario é acima de 1500 e não é homem?
.

- ternário (junta 3 operandos)
	- ?
	- :
	- teste ? true : false
	- média >=  7.0 ? "Aprovado" : "Reprovado"
